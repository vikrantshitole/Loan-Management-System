# Loan Management Frontend

React SPA for the Loan Management System.

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

Runs at `http://localhost:5173` and expects the API at `http://localhost:5000/api`.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server |
| `npm run build` | Production build |
| `npm run lint` | Run Oxlint |

See the root [README](../README.md) for the full project overview.
